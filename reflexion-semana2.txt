¿Por qué es importante para un desarrollador saber usar la terminal para navegar y organizar archivos dentro de un proyecto?
Aprender a usar la terminal es importante porque ayuda a trabajar de una forma más rápida y organizada dentro de un proyecto. Aunque al principio los comandos pueden parecer difíciles, poco a poco se vuelven muy útiles para crear carpetas, mover archivos y manejar mejor el trabajo. Además, muchas herramientas de programación funcionan desde la terminal, por eso es necesario familiarizarse con ella. También permite usar Git de una manera más práctica para guardar cambios y subir trabajos a GitHub. Considero que saber usar la terminal facilita mucho el trabajo de un desarrollador y ayuda a entender mejor cómo funciona el sistema del computador. Esta actividad me sirvió para practicar comandos básicos y sentirme más cómodo trabajando desde la consola.
